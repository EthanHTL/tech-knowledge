## Nodejs



### 多版本管理 nvm

```sh
# 查看可安装版本
nvm ls-remote

# 只看LTS版本
nvm ls-remote --lts
nvm ls-remote --lts=argon
# 查看当前版本
nvm current

# 查看当前已安装版本
nvm ls

# 切换版本
nvm use v18.20.3
```
