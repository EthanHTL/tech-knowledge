# Mac 常见问题



## 端口查看关闭

```bash
lsof -i :port   或     lsof -i tcp:port
kill -9 PID
```

## Parallals Desktop 无法启动

>  关闭 prl_disp_service



## 浏览器私密链接无法打开

> 键盘输入： thisisunsafe



## 破解软件损坏问题

```sh
$ sudo xattr -rd com.apple.quarantine  app路径
```
