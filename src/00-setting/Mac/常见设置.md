## 编程字体推荐

[Fira Code : Github](https://github.com/tonsky/FiraCode)

```sh
# MacBook 安装
brew install font-fira-code


# vs code 安装 
# settings.json 文件配置
"editor.fontFamily": "Fira Code",
"editor.fontLigatures": true,
```
